# fd スキル

fdはfindコマンドのシンプルで高速な代替ツールです。

## 主な機能

- 高速なファイル名検索
- .gitignore対応
- 正規表現サポート
- カラー出力

## 使用例

```bash
# パターンに一致するファイルを検索
fd "pattern"

# 特定の拡張子で検索
fd -e md

# ディレクトリのみ検索
fd -t d "pattern"

# ファイルのみ検索
fd -t f "pattern"

# 隠しファイルも含める
fd -H "pattern"

# 絶対パスで表示
fd -a "pattern"
```

Bashツールを使用してfdコマンドを実行してください。

## 注意事項

以下の操作はコマンド実行を伴うため、実行前にユーザーへ確認してください：

- `fd -x` / `fd --exec` - 検索結果に対してコマンドを実行するオプション
- `fd -X` / `fd --exec-batch` - 検索結果をまとめてコマンドに渡すオプション

特に削除操作（`rm`）や変更操作を行うコマンドを実行する場合は、必ずユーザーの確認を得てください。
